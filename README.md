# Machine Learning Comprehensive Guides

A complete collection of Jupyter notebooks covering fundamental and advanced machine learning concepts with visual explanations and practical examples.

## 📚 Notebook Collection

### Core Concepts (Foundation)
- **VECTORS_SCALARS_DERIVATIVES_VISUAL_GUIDE.ipynb** - Linear algebra and calculus fundamentals with visualizations
- **REGULARIZATION_LOSS_FUNCTIONS_VISUAL_GUIDE.ipynb** - Loss functions, penalties, and bias-variance tradeoff

### Numpy & Basics
- **01_Numpy_Refresher_Part_1.ipynb** - NumPy fundamentals
- **02_Numpy_Refresher_Part_2.ipynb** - Advanced NumPy operations
- **03_Numpy_Refresher_Part_3.ipynb** - NumPy applications

### Regression Models
- **LINEAR_REGRESSION_COMPLETE_LEARNING_GUIDE.ipynb** - Linear regression with all enhancements (L1/L2, visualization, hypothesis testing)
- **BEGINNERS_BOOK_Linear_Regression_Simple.ipynb** - Simplified linear regression for beginners

### Classification Models
- **LOGISTIC_REGRESSION_COMPLETE_GUIDE.ipynb** - Logistic regression with regularization and ROC curves
- **KNN_CLASSIFIER_GUIDE.ipynb** - K-Nearest Neighbors for classification

### Clustering Algorithms
- **KMEANS_CLUSTERING_GUIDE.ipynb** - K-Means clustering with visualization
- **HIERARCHICAL_CLUSTERING_GUIDE.ipynb** - Hierarchical clustering methods
- **DBSCAN_CLUSTERING_GUIDE.ipynb** - Density-based clustering
- **GAUSSIAN_MIXTURE_MODELS_GUIDE.ipynb** - Gaussian Mixture Models

### Tree-Based Methods
- **DECISION_TREES_COMPLETE_GUIDE.ipynb** - Decision trees with entropy and Gini impurity
- **RANDOM_FOREST_COMPLETE_GUIDE.ipynb** - Random forests and feature importance
- **GRADIENT_BOOSTING_XGBOOST_GUIDE.ipynb** - Gradient boosting and XGBoost

### Advanced Topics
- **NEURAL_NETWORKS_GUIDE.ipynb** - Neural networks with backpropagation
- **KNN_REGRESSOR_GUIDE.ipynb** - K-Nearest Neighbors for regression
- **08_Linear_Regression_Coefficients.ipynb** - Coefficient interpretation

### Utilities & Comparisons
- **ENHANCED_DEMO_ALL_FEATURES.ipynb** - Demonstration of all features across algorithms
- **Data_Structures_Memory_Performance_Comparison.ipynb** - Memory and performance analysis
- **Gradient_Descent.ipynb** - Optimization techniques
